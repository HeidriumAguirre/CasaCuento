# CasaCuento Hotel & Spa — Landing Page

Sitio web promocional oficial del Hotel Boutique CasaCuento, ubicado en Mantagua, Concón, V Región, Chile.

🌐 **Sitio en vivo:** [https://casacuento.netlify.app](https://casacuento.netlify.app)
📦 **Repositorio:** [github.com/HeidriumAguirre/CasaCuento](https://github.com/HeidriumAguirre/CasaCuento)

---

## 📋 Descripción

Landing page moderna, elegante y 100% responsive (mobile-first) construida en un único archivo `index.html` con CSS3 y JavaScript embebidos. Sin dependencias externas, sin backend, optimizada para smartphones, tablets y escritorio.

### Secciones incluidas

- ✅ **Header / Navbar** con scroll effect, menú hamburguesa mobile y ScrollSpy
- ✅ **Hero** con fondo visual, badge dorado, CTAs duales
- ✅ **Servicios / Amenities** — 6 tarjetas (Piscina, Tinajas, Zonas verdes, Habitaciones, Estacionamiento, Wi-Fi)
- ✅ **Galería** con grid masonry y lightbox funcional
- ✅ **Ubicación** con Google Maps Embed
- ✅ **Footer** con redes sociales (Instagram, Facebook, Booking, Expedia)
- ✅ **Botón flotante de WhatsApp** con animación pulse

---

## 🛠️ Stack técnico

- **HTML5** semántico (`<header>`, `<main>`, `<section>`, `<footer>`, `<nav>`)
- **CSS3** moderno con Custom Properties, Flexbox, Grid, animaciones
- **JavaScript** vanilla (sin frameworks, < 2KB de JS)
- **Google Fonts:** Playfair Display + DM Sans
- **Accesibilidad:** WCAG AA, ARIA labels, skip link, `prefers-reduced-motion`

## 🚀 Despliegue

El sitio está desplegado automáticamente en **Netlify** desde este repositorio de GitHub:

- **Cada `git push` a `main` dispara un nuevo deploy automático**
- **Branch previews:** cada PR genera una URL temporal de preview
- **HTTPS** y **CDN global** incluidos

### URL de producción

🔗 [https://casacuento.netlify.app](https://casacuento.netlify.app)

---

## ✏️ Cómo actualizar el sitio

1. Edita el archivo `index.html` localmente
2. Haz commit de los cambios:
   ```bash
   git add .
   git commit -m "Descripción del cambio"
   ```
3. Haz push a GitHub:
   ```bash
   git push origin main
   ```
4. Netlify detectará el push y desplegará automáticamente en ~30 segundos
